# 🐾 Acepto Mascotas

A pet-friendly platform created to help families find rentals, businesses, and services where pets are welcome.

## 🚀 Features

- Pet-friendly rental listings
- Responsive user interface
- Search and filtering experience
- Business/service directory
- Modern frontend architecture

## 🛠️ Tech Stack

React
Vite
JavaScript
Tailwind CSS
Git
Vercel

## 🎯 Project Goal

Promote a more pet-friendly culture by connecting people with places that accept pets as part of the family.
